# JICF Men's Fellowship

Helping men become faithful disciples of Jesus Christ who joyfully make disciples of others.

> "Come, follow Me, and I will make you fishers of men."
> — Matthew 4:19

## Purpose

This repository contains the living documents, discipleship resources, and leadership materials developed for JICF Men's Fellowship.

These documents are intended to evolve over time as we continue to learn, teach, and make disciples together.
